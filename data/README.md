# Data layout

- `raw/ciudades/`: Fuentes de ciudades (incluye `EC.txt`).
- `raw/jerarquia/`: Referencias de provincias/cantones/parroquias (PDF y carpeta para CSV externos).
- `output/ciudades/`: Archivos generados por el ETL (`ciudades_ec.csv`, `insert_ciudad.sql`).

Los scripts crean subcarpetas según sea necesario; estos archivos sirven como guía de estructura.
